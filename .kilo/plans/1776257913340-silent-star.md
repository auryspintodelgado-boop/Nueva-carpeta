# Plan: Recomendaciones para Mejorar el Sistema AURYS

## Objetivo
Proponer mejoras y nuevas funcionalidades para fortalecer el sistema de gestión de recursos humanos.

---

## 1. Mejoras de Seguridad (Alta Prioridad)

### 1.1 Autenticación y Sesiones
- Implementar expiración de sesión por inactividad (timeout de 30 min)
- Agregar verificación de contraseña en cambio de perfil
- Forzar cambio de contraseña en primer login
- Agregar log de auditoría de acciones de usuarios

### 1.2 Control de Acceso
- Implementar permisos granulares por acción (ver, crear, editar, eliminar)
- Agregar middleware de verificación de estado activo de usuario

---

## 2. Funcionalidades Nuevas (Media Prioridad)

### 2.1 Módulo de Reportes
- Exportación a Excel y PDF
- Gráficos interactivos con Chart.js
- Reportes programados (cron job)

### 2.2 Notificaciones
- Alertas por email para seguimientos pendientes
- Notificaciones in-app de evaluaciones próximas
- Recordatorios automáticos

### 2.3 Dashboard Mejorado
- Métricas en tiempo real
- Widgets configurables por usuario
- Indicadores visuales de productividad

### 2.4 Importación/Exportación
- Importar personas desde Excel
- Exportar lista de personas a CSV
- Backup automático de base de datos

---

## 3. Mejoras de UX/UI

### 3.1 Interfaz
- Temas claros/oscuros (dark mode)
- Diseño responsive para móviles
- Mejora de mensajes de error/éxito
- Loading states consistentes

### 3.2 Búsqueda y Filtrado
- Filtros avanzados en listados
- Búsqueda con autocompletado
- Paginación configurable

---

## 4. Mantenimiento y Calidad

### 4.1 Código
- Agregar pruebas unitarias (PHPUnit)
- Documentar controladores y modelos
- Eliminar código duplicado (Persona.php vs PersonaModel.php)
- Estandarizar nombres de archivos

### 4.2 Base de Datos
- Agregar índices para optimización
- Crear procedimientos almacenados para estadísticas
- Implementar soft deletes

---

## 5. Roadmap Sugerido

| Fase | Funcionalidades | Tiempo Estimado |
|------|-----------------|-----------------|
| 1 | Seguridad (auditoría, permisos, sesión) | 1 semana |
| 2 | Export/Import Excel, Dashboard mejorado | 1 semana |
| 3 | Notificaciones email, filtros avanzados | 1 semana |
| 4 | Pruebas unitarias, documentación | 1 semana |
